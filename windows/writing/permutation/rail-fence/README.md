# Shellcode Loader

Transform the shellcode with custom algorithm.

### Overview

Mengatur posisi byte dalam shellcode menggunakan algoritma rail-fence (zig-zag) cipher. 

Shellcode yang tersimpan memiliki format `[Key] [Encoded Shellcode]`.
