# Shellcode Loader

Executing shellcode as a fiber.

### Overview

Eksekusi shellcode dengan `RtlUserFiberStart`.

`RtlUserFiberStart` adalah fungsi internal dan bersifat low-level.

```c++
NTSTATUS RtluserFiberStart();
```
